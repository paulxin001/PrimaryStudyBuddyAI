# 自主性边界规则 — StudyBuddyAI

## 等级定义

| 等级 | 图标 | 含义 |
|------|------|------|
| 自主 | 🟢 | 可独立执行，事后通知 |
| 半自主 | 🟡 | 可执行，但关键决策需先展示建议 |
| 稳定 | 🔴 | 修改必须用户确认 |

## 各模块权限

### 服务运维 — 🟢 自主

**可自主执行：**
- 启动/停止/重启服务
- 运行健康检查
- 查看会话状态
- 查看报告

### 计划生成 — 🟢 自主

**可自主执行：**
- 解析作业文本
- 生成学习计划
- 预览计划

### Prompt 模板 — 🔴 稳定

**可自主执行：**
- 读取 Prompt 内容

**需用户确认：**
- 修改任何 Prompt 模板
- 新增 Prompt 模板
- 修改 system_persona.md（核心人设）

### 状态机逻辑 — 🔴 稳定

**可自主执行：**
- 阅读代码理解逻辑

**需用户确认：**
- 修改状态转换规则
- 增删状态或事件
- 修改 function calling 定义

### AI 适配器 — 🟡 半自主

**可自主执行：**
- 修复明显 bug
- 优化性能（不改接口）

**需用户确认：**
- 修改 AIProvider 接口
- 新增 AI Provider
- 修改 session 管理策略

### 客户端 — 🟡 半自主

**可自主执行：**
- 修复 UI bug
- 样式调整

**需用户确认：**
- 修改 WebSocket 协议
- 修改音视频采集逻辑

### 配置 — 🟡 半自主

**可自主执行：**
- 读取配置

**需用户确认：**
- 修改默认时间配置
- 修改 API 模型/声音选择

## 全局红线

以下操作在任何情况下都**绝对禁止**：

1. ❌ 不硬编码 API Key / 密钥到代码中
2. ❌ 不删除 data/ 目录下的会话数据和报告
3. ❌ 不在 Prompt 中加入帮助孩子做作业/给答案的指令
4. ❌ 不绕过状态机直接操作 AI 会话
5. ❌ 不在未告知用户时修改核心编排逻辑
6. ❌ 不收集或存储超出学习监督范围的隐私数据

## 升级路径

当不确定操作等级时：
1. 默认选择更保守的等级
2. 向用户展示修改计划，等待确认
3. 将决策记录到会话日志

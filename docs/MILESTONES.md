| 任务 | 状态 | 说明 |

## M1: 能用（第一次真实使用）

**通过标准：** 一个真实的孩子，用手机完成了一整次作业（至少 2 科），全程有 AI 语音陪伴，结束后生成了报告。

| 任务 | 状态 | 说明 |
|------|------|------|
| 火山 RTC 语音通路跑通 | ⬜ | 手机端能听到 AI 说话，AI 能听到孩子说话 |
| 计时器到点自动切换科目 | ⬜ | TaskBriefing → Studying → BreakTime 全流程 |
| 报告生成并展示在 Web 页面 | ⬜ | 家长打开 /report/{id} 能看到有意义的报告 |
| 手机端 PWA 可用 | ⬜ | 摄像头 + 麦克风权限、横屏适配、断线重连 |
| 全程录屏 | ⬜ | 作为短视频素材和产品迭代的依据 |

**验收方式：** 一段 ≥ 5 分钟的、未经剪辑的实际使用录屏。

**预估：1-2 周（在 M0 之后）**

**通过标准：** 发布至少 3 条短视频，其中 1 条播放量 ≥ 1 万。

|------|------|------|

**预估：1 周（在 M1 之后）**

## M3: 能留（10 个真实用户持续使用）

**通过标准：** 10 个家庭连续使用 5 天，留存率 ≥ 50%（即 ≥ 5 个家庭第 5 天还在用）。

| 任务 | 状态 | 说明 |
|------|------|------|
| 从 M2 的意向用户中筛选 10 个内测家庭 | ⬜ | 优先选一二年级、双职工家庭 |
| 行为数据结构化日志 | ⬜ | 每次 AI 判断 + 孩子反应 + 家长反馈存入 jsonl |
| 根据内测反馈迭代 Prompt 话术 | ⬜ | 按提醒次数/时段/科目分级，建立话术矩阵 |
| 误判率统计与优化 | ⬜ | 目标：分心提醒准确率 ≥ 70% |
| 家长满意度问卷 | ⬜ | 简单 5 道题：有用吗？孩子反感吗？愿意付费吗？ |

**验收方式：** 5 天留存数据 + 家长满意度评分 + 误判率统计报告。

**预估：2-3 周**

## M4: 能赚（第一笔收入）

**通过标准：** 月收入 ≥ 1000 元（无论来源）。

| 路径 A: To C 订阅 | 说明 |
|------|------|
| 免费版：每天 1 次，30 分钟 | 引流 |
| 付费版：无限次 + 详细报告 + 历史记录 | 39 元/月 或 298 元/年 |
| 微信支付/支付宝接入 | 最小化实现 |

| 路径 B: To 硬件 | 说明 |
|------|------|
| 把 AI 作业监督做成小智的"技能插件" | 联系小智生态 |
| 或直接联系 2-3 家教育硬件厂商 | 带着 M1 的演示视频去谈 |

**这两条路可以并行探索，哪条先跑通就先走哪条。**

**预估：1-2 个月（在 M3 之后）**
